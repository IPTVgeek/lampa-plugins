# IPTVGeek — плагины для Lampa

Коллекция плагинов для медиацентра [Lampa](https://lampa.mx/). Здесь собираются все плагины, которые я делаю.

## Плагины

### MediaInfo — дорожки и субтитры раздачи

Показывает аудиодорожки, субтитры и параметры видео (разрешение, HDR/Dolby Vision, кодек, битрейт) прямо в списке торрентов — до запуска плеера. Работает и в браузере, и на Android TV с внешним плеером.

**Папка:** [`mediainfo/`](mediainfo/)

**Установка** (Настройки → Расширения → Добавить):

```
https://raw.githubusercontent.com/IPTVgeek/lampa-plugins/main/mediainfo/plugin.js
```

Или через jsDelivr (для тестов лучше фиксировать конкретный коммит вместо `@main` — он кэшируется на сутки):

```
https://cdn.jsdelivr.net/gh/IPTVgeek/lampa-plugins@main/mediainfo/plugin.js
```

**Возможности:**

- двухслойные бейджи: мгновенно из названия (4K/1080p/720p, HDR10/HDR10+/Dolby Vision, кодек) + обогащение по ffprobe (точные аудио/субтитры, битрейты, Atmos, формат субтитров);
- источник ffprobe — локальный TorrServer (`/ffp`) с резервом на публичный анализатор;
- постоянный кэш, не патчит глобальные объекты (совместимо со strict mode).

**Как делался плагин** (весь процесс разработки): [статья](mediainfo/Как-я-делал-плагин-MediaInfo.md)

## Структура репозитория

```
mediainfo/
  plugin.js   — боевая версия
  debug.js    — отладочная версия с экранным логом (удобно дорабатывать на приставке)
  Как-я-делал-плагин-MediaInfo.md — статья о разработке
```

Каждый новый плагин — отдельная папка с `plugin.js` внутри.

## Для тех, кто хочет дорабатывать

Настройки вынесены в начало `mediainfo/plugin.js`: `AUTO_LIMIT`, `CONCURRENCY`, `HOST185` (адрес резервного анализатора), `DEBUG` (лог в консоль). PR и issue приветствуются.
